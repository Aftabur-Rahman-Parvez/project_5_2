Features

Dashboard

A complete dashboard that gives a complete overview of total sales, employees, items available etc. It also gives a summary of recent and current sales statistics.

Menu Item

The user may add, edit, and remove food items and their categories in this feature.

Inventory Management

The user is able to keep track of supplied goods and supplier lists from this feature.

Expense

The user can add expenses made in this module. They can also manage expense category from here.

Test List

This list gives you the information of all the available tests and their corresponding costs and description.

Sales

This feature gives a list of all the sold items till date with customer name, invoice numbers, paid amount etc.

Manage Employee

The user is able to see their employee details as well as their payments. The feature also allows the user to create access roles. The user shall also be able to track the employee login activities.

POS

This feature allows the user with expanded payment facilities as well as providing with real time inventory data. It provides you with detailed receipts with date and amount of sales.

Customer

The user may add customer details like email address, contact numbers from here.

Reports

Sales Report

The user can see the current and datewise sales information along with customer name, invoice numbers, paid amount etc.

Expense Report

The user can see the current and datewise expense information along with expense name, category, amount etc.
