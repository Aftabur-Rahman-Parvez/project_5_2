Features

Dashboard

The dashboard provides an overview of the total number of employees in the diagnostic hospital.

Patient List

User can add numerous patient profiles with their information like patient ID, blood group, prescribed medicines, medical histories, test lists etc.

Medical Store Attendant List

The user add a number of medical store attendant lists that has information like name, address, phone number, photo id etc.

Lab Attendant List

The user may add lab attendant list that has information like name, address, phone number, photo id etc.

Test List

This list gives you the information of all the available tests and their corresponding costs and description.

Doctor List

The user can add unlimited doctor name and their details like address, mobile number, qualification, department, registration number etc.

Receptionist List

The user can add unlimited receptionists and their details like address, mobile number, joining date, email address etc.

Patient Payment

You can save the patients’ payment history here with their contact details, medical history and test details.

Referral Agent List

The user can add lab attendant list that has information like name, address, phone number, email address, agent contact details and referred test list with patient id and their details.

Reports

Patient Report

You can see the patients’ over all test result information, their contact details, prescribed medicines and transaction details.

Test Reports

The report consists of all the test services provided and their payments corresponding to patient IDs.

Doctor Report

You can see the doctors’ contact information, their visiting hours and patient appointment details.

Referral Agent Report

This report provides information on commissions, tests provided over a timeline for a certain referral agent.
