Features

Dashboard

The dashboard provides an overview of the total number of employees in the diagnostic hospital.

Supplier List

The user can add or view unlimited supplier information like their address, mobile number, type of supplier date, email address etc. They also be able to pay the supplier and view all the transactions for a particular supplier from this module.

Returned Product List

The user is able to keep track of all the returned products having defect or any other issues.

Settings

This module allows the user to add categories, sub-categories, unit and service charges.

Manage Product

The user can add, view, delete product and modify their category from this module. They will also be to keep track of complete product stock and those which are running low. The user will be able to purchase product using this feature as well.

Manage User

User can create or delete access roles as required from this section.

Bank Transaction

In this module, the user is able to keep track of all the incoming and outgoing cash details. Transaction reports between specific dates can be generated from this section. They can also see the number of transactions along with bank details from here.

All Activity Log

This module is accessible to administrator to keep an eye on all the session logins.

Reports

Sales Report

The user will be able to generate reports by dates, executive access, customer or product wise. They can also process reports based on service charge and due amounts. Reports are also generated based on customer history.

Purchase Report

User is able to generate reports on purchases.
